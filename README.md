# YP-DS-01-Credit-scoring
<h2>Описание проекта</h2>
<b>Заказчик</b> — кредитный отдел банка. Нужно разобраться, влияет ли семейное положение и количество детей клиента на факт погашения кредита в срок. Входные данные от банка — статистика о платёжеспособности клиентов.<br>
  
Результаты исследования будут учтены при построении модели кредитного скоринга — специальной системы, которая оценивает способность потенциального заёмщика вернуть кредит банку.<br>

<h2>Описание данных</h2>
<h3>Пользователи</h3>
<ul>
<li><b>user_id</b> - уникальный идентификатор пользователя</li>
<li><b>name</b> - имя пользователя</li>
<li><b>age</b> - возраст</li>
<li><b>city</b> - город</li>
<li><b>subscription_type</b> - тип подписки (free, ultra)</li>
</ul>

<h3>Поездки</h3>
<ul>
<li><b>user_id</b> - уникальный идентификатор пользователя</li>
<li><b>distance</b> - расстояние, которое пользователь проехал в текущей сессии (в метрах)</li>
<li><b>duration</b> - продолжительность сессии (в минутах) — время с того момента, как пользователь нажал кнопку «Начать поездку» до момента, как он нажал кнопку «Завершить поездку»</li>
<li><b>date</b> - дата совершения поездки</li>
</ul>

<h3>Подписки</h3>
<ul>
<li><b>subscription_type</b> - тип подписки</li>
<li><b>minute_price</b> - стоимость одной минуты поездки по данной подписке</li>
<li><b>start_ride_price</b> - стоимость начала поездки</li>
<li><b>subscription_fee</b> - стоимость ежемесячного платежа</li>
</ul>
