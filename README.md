# YP-DS-01-Credit-scoring
<h2>Описание проекта</h2>
<b>Заказчик</b> — кредитный отдел банка. Нужно разобраться, влияет ли семейное положение и количество детей клиента на факт погашения кредита в срок. Входные данные от банка — статистика о платёжеспособности клиентов.<br>
  
Результаты исследования будут учтены при построении модели кредитного скоринга — специальной системы, которая оценивает способность потенциального заёмщика вернуть кредит банку.<br>

<h2>Описание данных</h2>
<ul>
<li><b>children</b> — количество детей в семье</li>
<li><b>days_employed</b> — общий трудовой стаж в днях</li>
<li><b>dob_years</b> — возраст клиента в годах</li>
<li><b>education</b> — уровень образования клиента</li>
<li><b>education_id</b> — идентификатор уровня образования</li>
<li><b>family_status</b> — семейное положение</li>
<li><b>family_status_id</b> — идентификатор семейного положения</li>
<li><b>gender</b> — пол клиента</li>
<li><b>income_type</b> — тип занятости</li>
<li><b>debt</b> — имел ли задолженность по возврату кредитов</li>
<li><b>total_income</b> — ежемесячный доход</li>
<li><b>purpose</b> — цель получения кредита</li>
</ul>
